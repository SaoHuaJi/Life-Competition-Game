# 生命竞争游戏 v0.1.0

这是一个基于康威生命游戏扩展出的单机可视化策略对战游戏。当前版本支持：

- 三种对局模式：`演示模式`、`人机对战`、`AI 对决`
- 三类地图：`方形`、`六边形`、`三角形`
- 三种主要胜利模式：`生存`、`全歼`、`占点`
- 多种族、多阵营、敌对伤害、天灾、额外繁衍、关键点占领
- 图样放置、图样选区、图样移动、模板导入导出、日志与回放
- 无限地图（宽高同时为 `0` 时启用，当前不支持占点模式）

## 1. 环境与启动

### 1.1 运行环境

项目使用：

- `React 19`
- `TypeScript 5`
- `Vite 7`

- `Node.js 20+`

- `Node.js 24.13.0`

```bash
npm install
```

```bash
npm run dev
```

构建生产版本：

```bash

```

本地预览生产构建：

## 2. 核心规则

### 2.1 基础演化

每个单元格在每一世代同步更新，单元格内要么为空，要么存在一个带生命值的细胞。

- **存活判定**：每个种族都有自己的 `存活范围`，若细胞周围友军数量落在该范围内，则继续存活。
- **出生判定**：每个种族都有自己的 `出生范围`，若空单元格周围满足某种族出生条件，则尝试在该格出生。
- **生命值恢复**：细胞满足存活条件后，会按本种族的 `恢复倍率` 恢复生命值，但不会超过最大生命值。
- **敌对伤害**：若启用敌对伤害，敌军会对目标细胞造成伤害；`对敌抗性` 会影响实际承伤。

### 2.2 地图、邻域与边界

支持三种网格：

- 方形地图
- 六边形地图
- 三角形地图

邻域决定“谁算邻居”，也决定三角地图与天灾范围的距离计算方式：

- `摩尔邻域`：共享边或共享点的单元格都算邻居
- `冯诺伊曼邻域`：仅共享边的单元格算邻居

- 六边形地图固定使用六边形邻域（冯诺伊曼邻域）

边界拓扑支持：

- `有界`：超出边界即不存在
- `环面`：左右、上下首尾相连

### 2.3 种族、阵营与特性

每个种族都具有独立参数：

- 名称、颜色、阵营编号
- 最大生命值
- 存活范围 / 出生范围
- 恢复倍率
- 天灾抗性 / 对敌抗性
- 初始细胞数
- AI 风格
- 种族特性

相同阵营的种族视为友军：

- 统计友军/敌军时按阵营判断
- 阵营排名会将同阵营种族的指标求和

当前实现中的特性及其效果：

- `无特性`：无特殊效果
- `长生`：最大生命值 `+3`，额外繁衍周期 `+3`
- `短寿`：最大生命值 `-3`、额外繁衍周期 `-3`，最终值至少为 `1`；额外繁衍数量乘 `1.25` 后向下取整
- `射手`：拥有独立于普通邻域的远程攻击范围
- `战士`：对敌对细胞伤害乘 `1.5` 后向上取整，受敌对细胞伤害乘 `0.75` 后向上取整

### 2.4 出生冲突

当同一空单元格同时满足多个种族的出生条件时，由“出生冲突策略”决定是否出生以及出生给谁。当前支持：

- `平局不出生`
- `随机`
- `友军数最多`
- `净优势最大`
- `优先顺序`

### 2.5 额外繁衍

若启用额外繁衍，则每个种族会按自己的繁衍周期进入繁衍阶段。

- 普通繁衍前提：该种族场上仍有活细胞
- 人类种族：由玩家手动提交新增位置
- AI 种族：自动提交候选位置
- 统一结算规则：已有细胞的单元格不能繁衍；若多个种族选择同一空格，该格不会产生新细胞

在允许补兵的模式中：

- `生存模式`
- `占点模式（允许复活）`

灭绝种族可以通过“复活补兵”重新获得初始细胞名额。复活成功后，会刷新该种族的繁衍等待回合计数。

### 2.6 天灾

天灾为按概率触发的环境事件。每轮可出现若干个落点，每个落点向外产生范围伤害。

当前实现规则：

- `天灾范围 = 1`：只有落点受伤
- `天灾范围 = 2`：距离落点为 `1` 的单元格也受伤
- 一般化为：`最远受影响距离 = 天灾范围 - 1`

若启用衰减，则距离越远伤害越低。

### 2.7 胜利模式

#### 生存模式

核心目标是尽量保留更多活细胞。

常见结束条件：

- 达到最大回合数
- 用户手动结束

排名规则：

1. 存活细胞数
2. 当前总生命值
3. 累计产生细胞数

#### 全歼模式

核心目标是消灭其它种族。

- 仅剩一个仍有活细胞的种族时可直接获胜
- 若启用“允许提前结束”，也可在到达最大回合时按排名结算

排名规则：

1. 存活回合数
2. 累计产生细胞数

#### 占点模式

地图上会生成关键点。细胞站在关键点上即可占领该点。

- 若某种族同时占有不少于 `占点目标数` 的关键点
- 并且这些用于凑足目标数的关键点保持连续占领达到 `占点维持回合`
- 则该种族获胜

若 `允许复活 = false` 且场上已无任何活细胞，游戏会立即结束并按当前排名结算。

排名规则：

1. 占点数
2. 存活细胞数
3. 当前总生命值
4. 累计产生细胞数

### 2.8 阵营排名

除种族排名外，系统还会计算阵营排名。阵营指标为同阵营所有种族对应指标的求和：

- 存活细胞数
- 总生命值
- 占点数
- 累计产生细胞数
- 存活回合数

## 3. 参数说明

界面右侧当前分为 `通用`、`规则`、`种族` 三页。

### 3.1 通用参数

#### 地图

- `地图类型`：决定单元格形状、可选邻域类型、距离计算方式和可用图样范围
- `宽度` / `高度`：设置地图尺寸；与宽高同时设为 `0` 时进入无限地图第一版
- `边界`：决定越过地图边缘后是消失还是从另一侧回到棋盘
- `邻域`：定义哪些格子算邻居，并影响存活、出生、距离、天灾和射手攻击范围

#### 对局

- `胜利模式`：决定对局采用生存、全歼、占点或纯演示结算规则
- `最大回合`：限制对局最长推进世代数；不同胜利模式会决定到达后是否立即结算

#### 日志与随机性

- `随机种子`：固定随机行为，便于复盘、复现和比较不同配置
- `布子限时`：限制开局布子阶段与繁衍阶段的可操作时长，单位为秒；取值为 `0` 时表示不限时
- `记录日志`：控制是否记录天灾、繁衍、胜负等对局日志

#### 天灾

- `启用天灾`：控制是否让环境天灾参与每一世代结算
- `触发概率`：决定每世代触发一轮天灾的概率
- `最少落点`：限制单轮天灾至少生成多少个落点
- `最多落点`：限制单轮天灾至多生成多少个落点
- `天灾范围`：决定单个落点能影响多远的图距离；范围值减一等于最远受影响距离
- `中心伤害`：定义天灾落点中心位置的基础伤害值
- `是否衰减`：控制伤害是否随距离增大而衰减
- `衰减系数`：定义每增加一层距离时伤害减少多少

#### 场景与模板

- `场景预设`：快速载入系统内置的参数场景与推荐玩法组合。
- `场景参数模板`：导入或导出当前通用参数与规则参数，便于复用整套规则配置
- `图样预设与模板`：选择预设图样、导入图样模板文件，或把当前选区导出为图样模板文件

### 3.2 规则参数

#### 出生与战斗

- `启用敌对伤害`：控制不同阵营细胞之间是否会在相邻或射手攻击范围内互相造成伤害
- `出生冲突策略`：决定多个种族同时满足同一空格出生条件时如何裁决归属

#### 额外繁衍

- `启用额外繁衍`：控制是否启用周期性繁衍阶段
- `允许公开布子位置`：决定参与游戏的种族在开局布子阶段与繁衍阶段是否能实时看到其它种族提交的位置；观测者始终可见全部位置
- `繁衍周期`：定义每个种族从最近一次初始布置或复活后，间隔多少世代进入一次繁衍阶段
- `繁衍数量`：定义每次繁衍阶段可新增的细胞数量或数量系数

#### 占点模式专有

- `关键点数量`：设置地图上生成多少个关键点
- `占点目标数`：定义同时占领多少个关键点才算开始冲击占点胜利
- `占点维持回合`：定义满足目标占点数后，需要持续保持多少回合才获胜
- `关键点生成方式`：决定关键点由系统随机生成，还是由玩家在布子阶段手动摆放
- `允许复活`：决定占点模式下灭绝种族能否在后续以复活补兵方式重新回到战场
- `允许提前结束`：决定占点模式是否允许在达到最大回合后直接按排名结算

#### 全歼模式专有

- `允许提前结束`：决定全歼模式是否允许在达到最大回合后按排名提前结算

#### 生存模式专有

- 生存模式无专有规则参数，主要依赖最大回合结算：生存模式主要比较回合结束时的存活规模，因此核心控制项是最大回合

### 3.3 种族参数

#### 基础参数

- `名称`：显示在界面、日志、统计和排名中的种族名称
- `颜色`：决定该种族在棋盘、统计卡片和图例中的主色
- `初始细胞数`：限制开局布子上限，也会影响允许复活时的补兵规模

#### 高级参数

- `最大生命值`：定义该种族细胞满血时的生命值上限
- `存活下限 / 存活上限`：共同定义细胞在多少个友军邻居包围下可以继续活到下一世代
- `出生下限 / 出生上限`：共同定义空格在多少个友军邻居支持下可以出生新细胞
- `恢复倍率`：定义细胞满足存活条件时，每世代恢复时使用的生命恢复倍率
- `天灾抗性`：定义该种族面对天灾时的承伤缩放程度，数值越高，实际受到的天灾伤害越低
- `对敌抗性`：定义该种族面对敌对细胞攻击时的承伤缩放程度，数值越高，实际受到的敌对伤害越低
- `种族特性`：为该种族附加战士、射手、短寿、长生等特殊规则修正
- `AI 风格`：决定该种族由 AI 控制时更偏向扩张、激进、防守、控点或随机

#### 模板

- `种族参数模板`：导入已有种族配置模板，或将当前种族参数导出为模板文件

### 3.4 棋盘显示与辅助参数

顶部和棋盘区域还提供若干显示辅助项：

- `单元尺寸`：调节棋盘单元格显示大小，方便观察局部或全局
- `推演间隔`：控制自动推进时每一世代的播放节奏
- `显示 HP`：控制是否在棋盘上显示细胞生命值信息
- `显示网格`：控制是否显示棋盘网格边线
- `邻域参考`：以鼠标当前位置为中心，显示当前邻域规则下的邻居范围
- `天灾范围预览`：以鼠标当前位置为中心，显示当前天灾范围会覆盖到哪些单元格
- `射手范围预览`：以鼠标当前位置为中心，显示当前地图与邻域规则下射手的攻击范围
- `居中视图`：将当前棋盘视图重新居中，便于在拖拽或缩放后快速回到主视角

## 4. 游玩方法

### 4.1 选择对局模式

顶部可切换：

- `演示模式`：适合观察规则和调参
- `人机对战`：默认第一种族为玩家，其余为 AI
- `AI 对决`：全部种族由 AI 控制

### 4.2 调整参数

在右侧面板中按需设置：

- 地图与邻域
- 胜利模式
- 天灾
- 额外繁衍
- 种族与阵营参数
- 模板与场景

若只想快速开始，可直接使用默认配置或预设。

### 4.3 布子阶段

点击 `重新开始` 后会清空棋盘并回到第 `0` 世代布子阶段。

当前布子支持：

- `编辑细胞`：左键按住连续绘制细胞，右键按住连续擦除细胞
- `选择图样`：左键点选或拖拽框选，右键点取消或拖拽框取消
- `放置图样`：选择图样后进入放置模式，左键确认放置，右键取消放置
- `移动选中图样`：将当前选中的图样整体移动到新位置，目标位置不能与未选中已有细胞冲突
- `编辑关键点`：仅在占点模式且关键点生成方式为手动时可用

### 4.4 开始与推进游戏

顶部主要控制按钮：

- `自动播放`
  - 连续推进世代
- `单步推进`
  - 每次推进一个世代
- `暂停游戏`
  - 暂停自动推进
- `结束游戏`
  - 提前结束当前对局并结算
- `重新开始`
  - 保留当前参数，清空棋盘并回到布子阶段

### 4.5 繁衍阶段

当启用额外繁衍且到达对应种族繁衍回合时，会进入繁衍阶段。

人类玩家可在棋盘上提交新增位置：

- 左键提交新增位置
- 撤销最近一次提交
- 删除已提交的候选位置
- 点击结束繁衍阶段，统一结算

AI 会自动提交自己的候选位置。

### 4.6 查看对局信息

对局过程中可以查看：

- 当前世代
- 每个种族统计
- 阵营统计
- 实时日志
- 回放滑条
- 天灾落点
- 邻域、天灾范围、射手攻击范围预览

### 4.7 模板文件使用

当前模板以文件方式导入导出，不在系统内做文件级重命名和删除。

可用模板包括：

- 图样模板
- 场景参数模板
- 种族参数模板

常见流程：

1. 先在界面中配置好参数或选中图样
2. 导出为模板文件
3. 以后通过导入模板文件复用
